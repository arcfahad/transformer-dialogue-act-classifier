# Transformer Dialogue Act Classification

This repository contains a Transformer encoder implemented from scratch
for dialogue act classification.

## Features
- Word2Vec static embeddings
- Positional encoding
- Multi-head self-attention
- Masked mean pooling
- Class imbalance handling
- Train / Validation / Test evaluation

## Files
- DataPreprocessing.ipynb
- Model_&_training.ipynb
- EncoderArchitecture.ipynb

Large files (Word2Vec, pickle files, checkpoints) are excluded.
